

# Kurly Clone Cording 팀프로젝트
## ⚙️ 설치 라이브러리 
### client
```bash
npm i
npm i react-bootstrap
npm i react-slick --save
npm i slick-carousel --save
npm i antd
npm i react-daum-postcode
```

### server
```bash
npm init --yes
npm i express --save
npm i mysql2
npm i cors
npm i nodemon --save-dev
npm i
```
